NIO，即none-blocking I/O（非阻塞I/O），和BIO是相对的存在，BIO称为blocking I/O即阻塞I/O。其实我自己理解中的NIO的N并不单单只是none的意思，还有new的意思，就是指NIO是区别于传统阻塞I/O的一种全新的I/O方式。
